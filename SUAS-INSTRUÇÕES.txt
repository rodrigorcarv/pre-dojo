##### CS - Rank #####

O projeto trata-se de um sistema de leitura de log e formação de rank de jogadores. 
O sistema foi desenvolvido com objetivo de estimula a competitividade dos jogadores, provendo estatística para 
comparação de desempenho dos jogadores.


##### Arquitetura ##### 

Para implementação do sistemas de rank criei uma arquitetura simples e eficiente utilizando spring-boot o que minimizar o tempo gasto com a configuração do projeto. E Visando atender melhor aos requisitos do problema proposto a solução não foi realizado utilizando persistência dos dados em banco ou em memória, devido ao curto tempo para desenvolvimento.


#### Tecnologias Utilizadas ####

- [Java 8]		- Foram um utlizadas nova biblioteca de Data e Stream para melhor desempenho, segurança e legibilidade do código fonte 
- [Maven]		- Maven para gerenciar as dependências e automatizar builds.
- [Spring]		- Framework MVC mais conhecido no mercado.
- [Spring Boot]	- Framework para desenvolvimento ágil de projetos;
- [JUnit]		- Ferramenta de testes amplamente utilizada.
- [Log4J]		- Ferramenta de log utilizada por todos os frameworks
- [REST Assured]- Framework para realização de testes utilizando chamadas REST
- [Mokito]      - Mokito utilizado para auxiliar a construção dos teste unitários.  

##### Pré-Requisitos##### 

Java 8
Maven

##### Instalação ##### 

Para instalação e necessário realizar o download do código-fonte git clone ](Link URL). Após realizar o download do código-fonte acesse a pasta do sistema e execute os comandos abaixo:

mvn clean install
mvn spring-boot:run
